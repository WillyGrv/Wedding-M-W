# Giona demo reference (Squarespace)

Source: https://giona-demo.squarespace.com/?password=avela

This demo showcases an elegant wedding site with:

- Typography: high-contrast serif for headings and a clean sans-serif for body.
- Visual identity: black backgrounds with refined illustrations, large imagery, generous whitespace, and centered titles.
- Sections: hero with couple names/date/place, story/itinerary (schedule), RSVP, registry, travel & stay, FAQs.
- Decorative assets: line-art illustrations, classic motifs.

Fonts (approximate, publicly available equivalents):
- Headings: Cormorant Garamond (serif, elegant, display-weight)
- Body: Inter (neutral sans-serif)

We’ll wire these into the project via Google Fonts and set CSS variables:
- `--font-heading: 'Cormorant Garamond', serif`
- `--font-body: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`

Notes:
- If you prefer the exact demo font (Squarespace may use Adobe Fonts/Typekit), share the kit URL and we’ll integrate it. For now, we use stylistically similar Google Fonts.
- We’ll keep existing Fraunces/Nunito Sans as fallbacks.

Next alignment ideas (optional):
- Increase title sizes and letter-spacing for hero headings.
- More contrast (dark-on-light or light-on-dark blocks) with decorative separators.
- Schedule section with time slots and line-art icons.
- Registry and Travel/Stay blocks styled as cards on neutral backgrounds.
